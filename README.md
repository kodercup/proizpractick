## Описание 

<br>
Данный продукт разработан в учебных целях. Веб-приложение для поиска и бронирования отелей позволяет пользователям просматривать каталог, сортировать по цене, регистрироваться, добавлять отели в избранное и управлять профилем. Проект реализован на HTML/CSS/JavaScript с использованием Local Storage и адаптивного дизайна.
<br>

## Установка
<br>
Для установки программы нужно зайти в раздел releases и скачать index.html  из самого нового релиза, после чего найти скачанный файл в своём файловом менеджере и нажать на него.После чего скопировать код и вставить его в текстовый редактор и запустить сервер.
<br>

## Использование 
<br>
Данное веб-приложение позволяет пользователям работать с каталогом отелей. При первом использовании доступна регистрация аккаунта или вход в существующий. Приложение предоставляет функции просмотра отелей с фото и ценами, сортировки по стоимости, добавления в избранное и управления профилем. Данные хранятся локально в браузере.
<br>

## Разработка 
<br>
Данный проект разрабатывается в рамках учебной практики одним человеком и не требует посторонней помощи. Веб-приложение создаётся с использованием чистого HTML, CSS и JavaScript для демонстрации навыков фронтенд-разработки, работы с DOM и управления состоянием приложения.
